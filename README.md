Installations
-------------

* Ruby version: 2.1.2

* Run Specs suite:
<pre>rake spec</pre>

* Run Specs with coverage:
<pre>COVERAGE=true rake spec</pre>

* Deployment instructions
<pre>git push heroku master</pre>
